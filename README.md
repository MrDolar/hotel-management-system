# Hotel Management System

AI-Powered Hotel Management Platform

## Features
- Room management (CRUD, types, pricing)
- Booking management (create, check-in, check-out)
- Guest management
- AI smart pricing suggestions
- AI occupancy prediction
- AI customer service chatbot

## Quick Start
```bash
cd backend
pip install -r requirements.txt
cp .env.example .env
uvicorn app.main:app --reload
```

API Docs: http://localhost:8000/docs
